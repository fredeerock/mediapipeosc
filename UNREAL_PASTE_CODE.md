# Copy-Paste Blueprint Code for Unreal Engine

This file contains actual blueprint code you can copy and paste directly into Unreal Engine.

---

## INSTRUCTIONS:

1. **Open Unreal Engine** and create your Blueprint Actor
2. **Open the Event Graph**
3. **Copy the code blocks below** (each section separately)
4. **Paste directly into the Event Graph** (Ctrl+V or Cmd+V)
5. The nodes will appear in your blueprint!

---

## USAGE INSTRUCTIONS:

### Step 1: Enable OSC Plugin
1. Edit → Plugins → Search "OSC"
2. Enable the OSC plugin
3. Restart Unreal Engine

### Step 2: Create Blueprint
1. Content Browser → Right-click → Blueprint Class → Actor
2. Name it `BP_OSC_Receiver`
3. Open the blueprint

### Step 3: Paste Code Blocks
1. Open Event Graph
2. Copy **CODE BLOCK 1** above
3. Paste into Event Graph (Ctrl+V)
4. Repeat for CODE BLOCKS 2-6

### Step 4: Connect the Nodes
After pasting, you'll need to connect:
- **BeginPlay** "then" → **CreateOSCServer** "execute"
- **CreateOSCServer** "ReturnValue" → Save to a variable (promote to variable named "OSCServer")
- **OSCServer** variable → Bind to **OnOSCMessageReceived** event
- **OnOSCMessageReceived** "Message" → **GetAllFloats** "Message"
- **GetAllFloats** "Values" → **GetArrayItem** (X, Y, Z) "Array" pins
- **GetArrayItem** outputs → **MakeVector** X, Y, Z inputs
- **MakeVector** "ReturnValue" → **PrintString** or your game logic

### Step 5: Configure
- Change the **Port** to `8000` in CreateOSCServer node
- Set **ReceiveIPAddress** to `0.0.0.0`
- Set **bMulticastLoopback** to `true`
- Set **bStartListening** to `true`

### Step 6: Test
1. Place the actor in your level
2. Press Play
3. Start the Electron app (`npm start`)
4. Begin tracking
5. You should see "Position received!" messages in Unreal

## ALTERNATIVE: SIMPLER APPROACH

If copy-pasting doesn't work in your Unreal version, you can manually create these nodes:

1. Right-click → Add Event → Event BeginPlay
2. Right-click → OSC → Create OSC Server (set port to 8000)
3. Right-click → Add Custom Event → name it "OnOSCMessageReceived"
4. Add inputs: Message (OSC Message), IPAddress (String), Port (Int)
5. Right-click → OSC → Get All Floats
6. Right-click → Array → Get (copy 3 times for indices 0, 1, 2)
7. Right-click → Math → Make Vector
8. Right-click → String → Print String

Then connect them as shown in `UNREAL_BLUEPRINT_CODE.md`!
